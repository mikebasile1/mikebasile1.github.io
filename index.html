Introduction

Completing the Computer Science program and building this ePortfolio have helped me grow both technically and professionally. Over the past two years, I’ve gained valuable hands-on experience that has prepared me for a career in software engineering. Working on group projects taught me how to collaborate effectively using tools like GitHub and Agile methods. Sharing responsibilities and coordinating with others improved my communication skills and helped me learn how to clearly explain technical concepts, especially when gathering or presenting requirements.

My understanding of data structures and algorithms deepened through assignments that required building efficient solutions using trees, hash maps, and sorting techniques. I also applied Big-O analysis to optimize performance. In software engineering courses, I created full-stack projects using Java, JavaScript, and Python, and integrated databases like MySQL and SQLite. I learned how to follow coding standards, write maintainable code, design schemas, and implement CRUD operations. Security was another major focus, where I gained experience writing secure code, validating input, and preventing vulnerabilities such as SQL injection.

This program has given me the confidence and skills to succeed in the field of computer science. Creating this ePortfolio has allowed me to reflect on my growth and share the knowledge and values that I bring to future opportunities.
